# 梦落Bot
****
此项目基于**[绪山真寻Bot](https://github.com/HibiKier/zhenxun_bot)** 修改，基于 Nonebot2 和 go-cqhttp 开发，以 postgresql 作为数据库的QQ群娱乐机器人
## 关于
本项目是经过自身需求修改绪山真寻Bot后的项目，主体项目和绪山真寻Bot并未有大的变化
本项目遵循GPL-V3开源协议，禁止商用
如果有问题或建议，可以发送issues
和绪山真寻Bot同步更新

## 声明
如有侵权，请联系我删除

此项目仅用于学习交流，请勿用于非法用途

# 配置gocq

在 https://github.com/Mrs4s/go-cqhttp 下载Releases最新版本，运行后选择反向代理，
  后将gocq的配置文件config.yml中的universal改为universal: ws://127.0.0.1:8080/cqhttp/ws

# 获取代码
git clone https://github.com/meng-luo/mengluo_bot.git

# 进入目录
cd mengluo_bot

# 安装依赖
pip install -r requirements.txt

# 进行基础配置
####请查看 配置 部分####

# 开始运行
python bot.py
